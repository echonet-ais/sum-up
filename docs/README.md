## 📊 프로젝트 개요

**SumUp**은 Vibe Coding Hackathon을 위한 프로젝트 관리 및 이슈 트래킹 시스템입니다.

### 주요 기능
- 프로젝트 관리
- 이슈 트래킹
- 칸반 보드
- 대시보드
- AI 기능
- 실시간 협업

### 기술 스택
- Next.js 16.0.3
- React 19.2.0
- TypeScript 5
- Tailwind CSS 4
- Zustand 5.0.8
- Supabase
